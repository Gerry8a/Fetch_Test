# Fetch_Test


## Features

- Developed in Kotlin.
- Uses Android Architecture Components.
- MVVM architecture.

## Requirements

- Display all the items grouped by "listId"
- Sort the results first by "listId" then by "name" when displaying.
- Filter out any items where "name" is blank or null.
